# Pr-veeksamen
repository for prøveeksamen

jeg kan starte med å fortelle litt om prosjektet mitt, jeg skal lage et helt simpelt "gjett tallet" spill som kjøres av flask, jeg er ikke helt sikker på om jeg skal bevege meg inn mot databaser enda. spillet har en slik funksjon at tallet er tilfeldig for hver gang du taster riktig tall, så da reseter tallets verdi og du kan spille på nytt når du har fått riktig tall i spillet.

jeg startet med å søke opp hvordan jeg kombinerer repositoryen min med vs code, altså i github så jeg kan gjøre ulike kommandoer

nå har jeg brukt litt KI til å få inn basicsa som jeg ikke husket, for eksempel noen kommandoer i terminalen, sånn at jeg kan oppdatere alt inn til git av det jeg gjør, også måtte jeg få en liten oppdatering på render template og hvilken funksjon det har.

jeg har gått litt gjennom det man vanligvis burde kunne bare for å få en liten gjennomgang, også skal jeg se på de videoene om python og flask som erik har ute.

Jeg bruker SQLite3 for å lagre antall forsøk hver gang spilleren vinner, slik at resultatene kan analyseres senere.
jeg har brukt KI til denne delen, jeg kunne litt fra før, altså kommandoene, SELECT, USERS, JOIN etc, men jeg prøver å få meg selv til å kunne litt av det, det er greit å ha det i koden min, men jeg vil helst ha noe jeg faktisk kan forstå, så jeg føler det blir litt bedre.
